# 麻雀点数計算スキャナー (Mahjong Scan & Calc)

🀄 **カメラや画像から麻雀の手牌をスキャンし、リアルタイムで点数や成立役を瞬時に計算するWebベースの便利ツールです。**

サーバーなしのクライアントサイドのみで高速に動作するため、GitHub Pagesなどの静的ホスティング環境へ簡単に配置して、PCやスマートフォンから即座に利用することができます。

---

## 主な機能

### 1. 📷 カメラ・画像スキャナー (画像認識)
- カメラビューに表示される14枚のガイドグリッドに合わせて牌を撮影するだけで、自動的に手牌を切り出して認識します。
- クライアントサイドでの画像解析（Sobelエッジ検出 ＋ ZNCCテンプレートマッチング）を採用しており、モデルデータの重いダウンロードなしで、オフラインでも超高速・軽量に動作します。
- スマートフォンやPCのカメラ入力のほか、ローカル画像のドラッグ＆ドロップによるスキャンにも対応しています。

### 2. 🎴 対話型手牌エディター & 手動牌選択
- スキャンで誤認識された牌や、手動で手牌を調整したい場合、該当する牌をタップするだけで、美しい牌選択シートから**一瞬で変更・修正可能**です。
- 萬子・筒子・索子・字牌の全34種に加え、赤ドラ（五萬・五筒・五索）の選択にも対応しています。

### 3. 🧮 状況に応じた正確な点数計算
- アガリ時の「翻数」「符数」「最終得点（親／子のツモ和了時の支払い内訳含む）」を詳細に算出。
- 役判定エンジンには、実績豊富で精度の高い `riichi` ライブラリを採用。
- **各種詳細設定に対応:**
  - **和了条件:** 自摸 (ツモ) / 栄 (ロン)
  - **対局風:** 場風 (東・南・西) / 自風 (東・南・西・北)
  - **ドラ:** ドラ表示牌・裏ドラ表示牌の自由な追加・編集
  - **状況・役フラグ:** 立直 (リーチ)、ダブル立直、一発、嶺上開花、槍槓、海底/河底

### 4. 📱 高級感のあるモダンレスポンシブUI
- 深みのあるグリーン（麻雀卓マット）とすりガラス効果（グラスモーフィズム）を取り入れた、高級感のあるダークテーマ。
- 立体的な3D風デザインの麻雀牌オブジェクトと滑らかなマイクロアニメーション。
- PCの大画面からスマートフォンの縦画面まで柔軟に最適化されるレスポンシブレイアウト。

## 技術スタック

- **コア**: HTML5 / JavaScript (ES6 Modules)
- **スタイリング**: CSS3 (Vanilla CSSによるフルカスタム、すりガラスデザイン)
- **ビルドツール**: Vite (超高速バンドラー)
- **点数計算エンジン**: `riichi` (npmライブラリ)
- **画像認識**: Canvas API + Grayscale / Sobelエッジフィルタ + ZNCC (ゼロ平均正規化相互相関) マッチング

## 開発環境のセットアップ

ローカルで起動して動作を確認・開発する方法です。

### 1. 依存関係のインストール
```bash
npm install
```

### 2. ローカル開発サーバーの起動
```bash
npm run dev
```
起動後、ブラウザで `http://localhost:5173/` にアクセスして動かすことができます。

### 3. 本番用ビルドの作成
```bash
npm run build
```
`dist` フォルダに最適化された静的ファイルが生成されます。

## GitHub Pages へのデプロイ方法

本プロジェクトはGitHub Pagesへワンコマンドで簡単にデプロイできるよう、あらかじめ設定されています。

### 1. GitHubリポジトリの設定
GitHub上でリポジトリ（例: `mahjong-recognition`）を作成し、ローカルリポジトリにリモートとして追加しPushします。
```bash
git remote add origin https://github.com/あなたのユーザー名/mahjong-recognition.git
git add .
git commit -m "Initial commit"
git branch -M main
git push -u origin main
```

### 2. デプロイコマンドの実行
以下のコマンドを実行します。自動的にプロジェクトがビルドされ、`dist` ディレクトリの内容が自動的に `gh-pages` ブランチに反映されて公開されます。
```bash
npm run deploy
```

デプロイ完了後、 `https://あなたのユーザー名.github.io/mahjong-recognition/` にてツールが公開されます。
